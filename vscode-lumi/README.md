# Lumi Language

VS Code support for the Lumi programming language.

Features:

- `.lumi` file detection and icon
- Syntax highlighting
- Comment and bracket handling
- A `print` snippet
- **Lumi: Run Current File** command
- Run button in the editor title
- <kbd>Ctrl+F5</kbd> to run on Linux/Windows or <kbd>Cmd+F5</kbd> on macOS

The run command expects `lumi` to be available on your `PATH`.
